----Programa de Fuerza Bruta para Contraseñas----
El programa implementa un algoritmo básico de fuerza bruta en Python para descifrar contraseñas. El obetivo es comprender como funcionan los ataques de fuerza bruta.
----- Requisitos -----
Se necesita VS Code o Google Collab donde se pueda ejecutar Python3 
Librerías ----> En este caso para hacer la grafica
----- Ejecucion -----
La ejecucion se realiza en un ambiente de python
-----Funcionamiento-----
El programa genera combinaciones posibles 
Cada combinacion posible se compara con la contrasena que se quiere adivinar 
Se cuenta el nunero de intentos de cada combinacion
Se mide el tiempo de ejecucion
-----Ejemplo de Salida-----
Encontrada: 4Z 2 caracter(es) -> 3200 intentos, 0.02 segundos
-----Conclusion------
Se puede evidenciar como funciona el uso de fuerza bruta para poder encontras posibles combinacion y obtener una contrasena. Aunque es un programa corto demuestra la importancia de utilizar las contrasenas seguras. 
¿qué pasa si la contraseña tiene 8+ caracteres y usa mayúsculas, números y símbolos?
La contrasena va a ser mas dificil de descifrar debido a que se aumenta la posibilidad de combinaciony podria tomar anos descifrar una clave
